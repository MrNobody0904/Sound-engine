🎧 SOUND ENGINE – Web Audio Equalizer & Visualizer

SOUND ENGINE is a modern browser-based audio player with a powerful 10-band equalizer, real-time frequency spectrum visualizer, waveform display, compressor, microphone input, and advanced audio controls — all built using the Web Audio API.

🚀 Features
🎚 10-Band Equalizer

32Hz – 16kHz frequency bands

Boost / Cut from -15dB to +15dB

Smooth Q-factor control

Real-time audio processing
